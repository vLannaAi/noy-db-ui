# Theme

The colors themselves are the family's `--nui-*` CSS variables — to **brand** the library, set them on
`:root`. The full reference lives in **@noy-db/ui › Design tokens**. This page covers the binding's job:
**driving dark/light at runtime**.

## Dark mode

A `data-theme="dark"` attribute on `<html>` swaps to the dark token set (shipped in `tokens.css`).
Drive it with `useTheme()`:

```ts
import { useTheme } from '@noy-db/ui-nuxt/core'
const { mode, resolved, set } = useTheme()   // mode: light|dark|system; resolved: light|dark
set('dark')
```

`system` follows `prefers-color-scheme`. Components carry **zero theme logic** — it's all tokens.
